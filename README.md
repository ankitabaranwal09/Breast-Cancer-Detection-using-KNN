This project implements a machine learning model to detect breast cancer using the K-Nearest Neighbors (KNN) algorithm. 
The model classifies tumors as malignant or benign based on features extracted from medical diagnostic data.
Early detection of breast cancer significantly increases survival rates, making predictive models like this highly valuable in healthcare.
This project uses the Breast Cancer Wisconsin Dataset.
Features include:
Radius
Texture
Perimeter
Area
Smoothness
Compactness
Symmetry
